Exercice 1: Fraction

Résoudre l'inéquation suivante :

\(\frac{5}{x} > 2\)

Solution détaillée

Étape 1 : Mettre l'inéquation sous la forme d'une fraction positive

On transforme l'inéquation pour avoir la forme "expression > 0" :

\(\frac{5}{x} > 2\)

\(\frac{5}{x} - 2 > 0\)

\(\frac{5 - 2x}{x} > 0\)

Étape 2 : Déterminer les valeurs qui annulent le numérateur et le dénominateur

Pour le numérateur : \(5 - 2x = 0\)

  • \(2x = 5\)
  • \(x = \frac{5}{2}\)

Pour le dénominateur : \(x = 0\)

Étape 3 : Établir le tableau de signe

Ces valeurs (0 et \(\frac{5}{2}\)) divisent l'axe des réels en trois intervalles : \(]-\infty ; 0[\), \(]0 ; \frac{5}{2}[\) et \(]\frac{5}{2} ; +\infty[\)

\(x\) \(-\infty\) \(0\) \(\frac{5}{2}\) \(+\infty\)
\(x\) \(-\) \(0\) \(+\) \(+\)
\(5 - 2x\) \(+\) \(+\) \(0\) \(-\)
\(\frac{5 - 2x}{x}\) \(-\) interdit \(+\) \(0\) \(-\)
Étape 4 : Conclusion

L'expression \(\frac{5 - 2x}{x}\) est strictement positive sur l'intervalle \(]0 ; \frac{5}{2}[\).

L'ensemble solution de l'inéquation \(\frac{5}{x} > 2\) est donc : \(S = ]0 ; \frac{5}{2}[\)

Exercice 2: Identité remarquable

Résoudre l'inéquation suivante :

\((x+3)^2 - 16 \geq 0\)

Solution détaillée

Étape 1 : Utiliser l'identité remarquable pour développer l'expression

\((x+3)^2 - 16 \geq 0\)

On utilise l'identité remarquable \((a+b)^2 = a^2 + 2ab + b^2\) :

\((x+3)^2 - 16 = x^2 + 2 \times 3 \times x + 3^2 - 16\)

\((x+3)^2 - 16 = x^2 + 6x + 9 - 16\)

\((x+3)^2 - 16 = x^2 + 6x - 7\)

Étape 2 : Factoriser l'expression

On cherche à factoriser \(x^2 + 6x - 7\) sous la forme \((x+a)(x+b)\).

Méthode pour factoriser un trinôme de la forme \(x^2 + px + q\) :

1. On cherche deux nombres \(a\) et \(b\) tels que :

  • \(a + b = p\) (la somme donne le coefficient de \(x\))
  • \(a \times b = q\) (le produit donne le terme constant)

2. Si on trouve de tels nombres, alors \(x^2 + px + q = (x+a)(x+b)\)

Dans notre cas, pour \(x^2 + 6x - 7\), on cherche \(a\) et \(b\) tels que :

  • \(a + b = 6\) (somme)
  • \(a \times b = -7\) (produit)

Puisque le produit est négatif (\(-7\)) et la somme positive (\(6\)), on sait que:

  • Un des nombres est positif (appelons-le \(a\))
  • L'autre est négatif (appelons-le \(b\))
  • Le nombre positif a une valeur absolue plus grande que le nombre négatif (puisque la somme est positive)

Les diviseurs de 7 sont 1 et 7. On teste les paires possibles:

  • \(1 \times (-7) = -7\) et \(1 + (-7) = -6\) ❌ (la somme n'est pas 6)
  • \(7 \times (-1) = -7\) et \(7 + (-1) = 6\) ✓ (la somme est bien 6)

Donc \(a = 7\) et \(b = -1\), ce qui nous donne:

\(x^2 + 6x - 7 = (x+7)(x-1)\)

Étape 3 : Résoudre l'inéquation

Notre inéquation devient :

\((x+7)(x-1) \geq 0\)

Pour que le produit de deux facteurs soit positif ou nul, il faut que :

  • soit les deux facteurs sont de même signe (tous les deux positifs ou nuls, ou tous les deux négatifs ou nuls)
  • soit au moins un des facteurs est égal à zéro

Nous devons déterminer les valeurs qui annulent ces facteurs :

  • \(x+7 = 0 \Rightarrow x = -7\)
  • \(x-1 = 0 \Rightarrow x = 1\)
Étape 4 : Établir le tableau de signe

Ces valeurs (-7 et 1) divisent l'axe des réels en trois intervalles : \(]-\infty ; -7[\), \(]-7 ; 1[\) et \(]1 ; +\infty[\)

\(x\) \(-\infty\) \(-7\) \(1\) \(+\infty\)
\(x+7\) \(-\) \(0\) \(+\) \(+\)
\(x-1\) \(-\) \(-\) \(0\) \(+\)
\((x+7)(x-1)\) \(+\) \(0\) \(-\) \(0\) \(+\)
Étape 5 : Conclusion

L'expression \((x+7)(x-1)\) est positive ou nulle sur les intervalles \(]-\infty ; -7]\), \([1 ; +\infty[\) et aux points \(x = -7\) et \(x = 1\) où elle s'annule.

L'ensemble solution de l'inéquation \((x+3)^2 - 16 \geq 0\) est donc : \(S = ]-\infty ; -7] \cup [1 ; +\infty[\)

Exercice 3: Fonction carré

Résoudre l'inéquation suivante :

\(x^2 - 5x + 6 > 0\)

Solution détaillée

Étape 1 : Factoriser l'expression

On cherche à factoriser le trinôme \(x^2 - 5x + 6\) sous la forme \((x+a)(x+b)\).

Méthode pour factoriser un trinôme de la forme \(x^2 + px + q\) :

1. On cherche deux nombres \(a\) et \(b\) tels que :

  • \(a + b = p\) (coefficient de \(x\))
  • \(a \times b = q\) (terme constant)

2. Si on trouve de tels nombres, alors \(x^2 + px + q = (x+a)(x+b)\)

Dans notre cas, pour \(x^2 - 5x + 6\), on cherche \(a\) et \(b\) tels que :

  • \(a + b = -5\) (coefficient de \(x\))
  • \(a \times b = 6\) (terme constant)

Puisque le produit est positif (\(6\)) et la somme négative (\(-5\)), \(a\) et \(b\) sont tous deux négatifs.

Les diviseurs de 6 sont : 1, 2, 3 et 6. On teste les paires possibles avec des signes négatifs :

  • \((-1) \times (-6) = 6\) et \((-1) + (-6) = -7\) ❌ (la somme n'est pas -5)
  • \((-2) \times (-3) = 6\) et \((-2) + (-3) = -5\) ✓ (la somme est bien -5)

Donc \(a = -2\) et \(b = -3\), ce qui nous donne :

\(x^2 - 5x + 6 = (x+(-2))(x+(-3)) = (x-2)(x-3)\)

Étape 2 : Résoudre l'inéquation

Notre inéquation devient :

\((x-2)(x-3) > 0\)

Pour que le produit de deux facteurs soit strictement positif, il faut que les deux facteurs soient de même signe (soit tous les deux positifs, soit tous les deux négatifs).

Nous devons déterminer les valeurs qui annulent ces facteurs :

  • \(x-2 = 0 \Rightarrow x = 2\)
  • \(x-3 = 0 \Rightarrow x = 3\)
Étape 3 : Établir le tableau de signe

Ces valeurs (2 et 3) divisent l'axe des réels en trois intervalles : \(]-\infty ; 2[\), \(]2 ; 3[\) et \(]3 ; +\infty[\)

\(x\) \(-\infty\) \(2\) \(3\) \(+\infty\)
\(x-2\) \(-\) \(0\) \(+\) \(+\)
\(x-3\) \(-\) \(-\) \(0\) \(+\)
\((x-2)(x-3)\) \(+\) \(0\) \(-\) \(0\) \(+\)
Étape 4 : Conclusion

L'expression \((x-2)(x-3)\) est strictement positive sur les intervalles \(]-\infty ; 2[\) et \(]3 ; +\infty[\).

L'ensemble solution de l'inéquation \(x^2 - 5x + 6 > 0\) est donc : \(S = ]-\infty ; 2[ \cup ]3 ; +\infty[\)

Exercice 4: Fonction cube

Résoudre l'inéquation suivante :

\(x^3 - 4x > 0\)

Solution détaillée

Étape 1 : Factoriser l'expression

On commence par factoriser l'expression en mettant \(x\) en facteur :

\(x^3 - 4x = x(x^2 - 4)\)

Dans le second facteur \(x^2 - 4\), on reconnaît une identité remarquable de la forme \(a^2 - b^2\).

Rappel : \(a^2 - b^2 = (a-b)(a+b)\)

Avec \(a = x\) et \(b = 2\), on a :

\(x^2 - 4 = x^2 - 2^2 = (x-2)(x+2)\)

Donc l'expression complète devient :

\(x^3 - 4x = x(x^2 - 4) = x(x-2)(x+2)\)

Étape 2 : Résoudre l'inéquation

Notre inéquation devient :

\(x(x-2)(x+2) > 0\)

Pour que le produit de trois facteurs soit strictement positif, il faut qu'il y ait un nombre pair de facteurs négatifs (soit 0, soit 2).

Nous devons déterminer les valeurs qui annulent ces facteurs :

  • \(x = 0\)
  • \(x-2 = 0 \Rightarrow x = 2\)
  • \(x+2 = 0 \Rightarrow x = -2\)
Étape 3 : Établir le tableau de signe

Ces valeurs (-2, 0 et 2) divisent l'axe des réels en quatre intervalles : \(]-\infty ; -2[\), \(]-2 ; 0[\), \(]0 ; 2[\) et \(]2 ; +\infty[\)

\(x\) \(-\infty\) \(-2\) \(0\) \(2\) \(+\infty\)
\(x\) \(-\) \(-\) \(0\) \(+\) \(+\)
\(x-2\) \(-\) \(-\) \(-\) \(0\) \(+\)
\(x+2\) \(-\) \(0\) \(+\) \(+\) \(+\)
\(x(x-2)(x+2)\) \(-\) \(0\) \(+\) \(0\) \(-\) \(0\) \(+\)
Étape 4 : Conclusion

L'expression \(x(x-2)(x+2)\) est strictement positive sur les intervalles \(]-2 ; 0[\) et \(]2 ; +\infty[\).

L'ensemble solution de l'inéquation \(x^3 - 4x > 0\) est donc : \(S = ]-2 ; 0[ \cup ]2 ; +\infty[\)
